# Primitives

The assistant runtime is built from core primitives that can evolve into a general AI agent platform.

## Core Primitives

- events: communication backbone of the system.
- reasoning: LLM-based decision engine.
- tools: capability-based integrations.
- memory: long-term knowledge storage.
- environment: ambient context signals.
- automation: proactive behavior triggers.
- policy: safety and permission controls.
- observability: debugging and traceability.
